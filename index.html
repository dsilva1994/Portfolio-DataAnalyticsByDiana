Unlocking The Impact of Weekday Duration Dynamics on Campaign Performance

This project focused on the analysis of marketing campaign performance for the Google Merch Store using an e-commerce dataset of Google Analytics for 1 November 2020 to 31 January 2021. Key insights were derived with data preparation in SQL, BigQuery, and Power BI, focusing on user engagement, conversion rate, and Return on Investment (ROI). These tools and techniques ranged from SQL for advanced data preparation to Power BI for KPI calculation and dynamic visualizations, analysis of metrics such as CTR, CPM, CPA, ROI, bounce rates, and session duration. Recommendations were given which would drive actionable optimization of campaign timing in efforts to improve user engagement and, by default, maximize ROI. Full details can be found in the attached dashboard and report.

Funnel Analysis: User Journey from Awareness to Conversion

This project addresses a decline in conversions observed by the sales team in an online sales funnel. Using SQL, BigQuery, and Excel, I analyzed user behaviour across key funnel stages (awareness, interest, desire, intent, conversion) to identify where users drop off. Key insights focused on improving the checkout process, encouraging add-to-cart actions, and optimizing conversion rates across different countries and devices. Recommendations were made to reduce friction, enhance user experience, and ultimately boost sales performance.
